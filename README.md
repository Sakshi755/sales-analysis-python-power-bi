# Sales Analysis Dashboard

## Project Overview
This project focuses on analyzing sales data using Python and creating an
interactive dashboard using Power BI. The goal is to understand sales
performance, profit trends, and category-wise insights.

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- Power BI
- Excel

## Key Analysis Performed
- Data cleaning and preprocessing using Python
- Handling inconsistent date formats
- Monthly sales trend analysis
- Category-wise and region-wise sales analysis
- Profit analysis

## Dashboard Features
- Total Sales and Total Profit KPIs
- Monthly Sales Trend
- Category and Sub-category Analysis
- Region-wise analysis using slicers

## Outcome
The dashboard helps in understanding sales performance and identifying
top-performing categories and profitable regions.
